How to set up a graph

You start by creating an empty div

<div id="holder">
</div>

Inside of this div, the graph will be implemented. While the id and class of the div is not important, it's important that the div's width is at least 340px, otherwise it will not render

Then in jaascript, create a new instance of a graph and set the dom of the div as the object to pass in

let holder = document.getElementById("id-of-div");

let graph = new Graph(holder);
graph.create();

Now when the graph has been created, you can start inserting data to the graph. To add multiple instances of data, you can set the data equal to an object, like this: 

graph.data = [
    {
        key: "10 Dec",
        value: 12
    },
    {
        key: "11 Dec",
        value: 8
    }
];

This will add two data points to the graphs. If you want to add onto this data set, you can use the function appendData(data) which takes a list of objects and adds it to the end of the data set 

let data = [
    {
        key: "12 Dec",
        value: 13
    }
];

graph.appendData(data);

It can also accept a single object of data. Notice that this will add the data att the end of the list, to add data after or before a specific data point, use AddBefore(data, key) or AddAfter(data, key), wher the key is the data already in the data set.
