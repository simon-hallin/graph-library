How to set up a graph

You start by creating an empty div

<div id="holder">
</div>

Inside of this div, the graph will be implemented. While the id and class of the div is not important, it's important that the div's width is at least 340px, otherwise it will not render

Then in jaascript, create a new instance of a graph and set the dom of the div as the object to pass in

let holder = document.getElementById("id-of-div");

let graph = new Graph(holder);
graph.create();
